# Lingis

## Installation

```bash
npm install -g @ionic/cli
git clone https://github.com/GytisKau/lingis.git
cd lingis
npm install
```


## Build Commands

```bash
ionic serve # builds and opens a web browser

# other commands:
ionic build # builds files into dist/ directory
ionic serve --external # same as serve, but also lets you open the app on a network with a phone
# Example:
#     Local: http://localhost:8100
#     External: http://10.69.144.44:8100 - enter this url with your phone
```
